# Setup Google Integration — ILP Academy Integrasi Google (Forms, Drive) menggunakan **OAuth 2.0 Refresh Token**. Cocok untuk akun Gmail biasa — tidak perlu Google Workspace. --- ## Langkah 1 — Buat Project Baru di Google Cloud Console 1. Buka https://console.cloud.google.com 2. Klik dropdown project di atas → **"New Project"** 3. Isi nama project (contoh: `ILP Academy 2026`) → **Create** 4. Pastikan project baru tersebut sudah dipilih (terlihat di header) --- ## Langkah 2 — Aktifkan APIs 1. Menu **APIs & Services → Library** 2. Cari dan klik **Google Forms API** → **Enable** 3. Cari dan klik **Google Drive API** → **Enable** --- ## Langkah 3 — Setup OAuth Consent Screen 1. Menu **APIs & Services → OAuth consent screen** (atau **Google Auth Platform → Audience**) 2. Klik **"Get started"** atau **"Configure consent screen"** 3. Isi: - App name: `ILP Academy` - User support email: email Google Anda - Developer contact email: email Google Anda 4. Klik **Save and Continue** 5. Di bagian **Scopes**, klik **"Add or Remove Scopes"**, tambahkan: - `https://www.googleapis.com/auth/forms.body` - `https://www.googleapis.com/auth/forms.responses.readonly` - `https://www.googleapis.com/auth/drive` 6. Klik **Update** → **Save and Continue** 7. Di bagian **Test users**, klik **"+ Add users"**, masukkan email Google Anda sendiri 8. **Save and Continue** → **Back to Dashboard** --- ## Langkah 4 — Buat OAuth 2.0 Client ID 1. Menu **APIs & Services → Credentials** (atau **Google Auth Platform → Clients**) 2. Klik **"+ Create Credentials" → "OAuth client ID"** 3. Isi: - Application type: **Web application** - Name: `ILP Backend` - **Authorized redirect URIs**: klik **"+ Add URI"** → masukkan: `https://developers.google.com/oauthplayground` 4. Klik **Create** 5. Catat **Client ID** dan **Client Secret** yang muncul (atau download JSON-nya) --- ## Langkah 5 — Generate Refresh Token via OAuth Playground 1. Buka https://developers.google.com/oauthplayground 2. Klik ikon **gear (⚙)** di kanan atas 3. Centang **"Use your own OAuth credentials"** 4. Isi **OAuth Client ID** dan **OAuth Client Secret** dari Langkah 4 → tutup panel 5. Di panel kiri **(Step 1 — Select & authorize APIs)**: - Di kotak **"Input your own scopes"** paling bawah, ketik: ``` https://www.googleapis.com/auth/drive https://www.googleapis.com/auth/forms.body https://www.googleapis.com/auth/forms.responses.readonly ``` - Klik **"Authorize APIs"** 6. Login dengan akun Google yang sama dengan test user di Langkah 3 → klik **Allow** 7. Di **(Step 2 — Exchange authorization code for tokens)**: - Klik **"Exchange authorization code for tokens"** 8. Salin nilai **Refresh token** (dimulai dengan `1//...`) dari panel kanan --- ## Langkah 6 — Simpan Secrets di Supabase 1. Buka **Supabase Dashboard → Settings → Edge Functions → Secrets** 2. Tambahkan (atau update) tiga secrets berikut: | Name | Value | |------|-------| | `GOOGLE_CLIENT_ID` | Client ID dari Langkah 4 | | `GOOGLE_CLIENT_SECRET` | Client Secret dari Langkah 4 | | `GOOGLE_REFRESH_TOKEN` | Refresh token dari Langkah 5 | > Hapus `GOOGLE_SERVICE_ACCOUNT_JSON` jika masih ada (tidak dipakai). --- ## Langkah 7 — Deploy Edge Functions Jalankan di terminal dari folder project: ```bash npx supabase functions deploy google-form-create npx supabase functions deploy google-form-schema npx supabase functions deploy google-form-responses npx supabase functions deploy google-drive-setup npx supabase functions deploy google-drive-upload npx supabase functions deploy google-drive-organize ``` Atau sekaligus: ```bash npx supabase functions deploy google-form-create google-form-schema google-form-responses google-drive-setup google-drive-upload google-drive-organize ``` --- ## Verifikasi Setelah selesai, buka halaman **Admin → Materi** di app. Jika setup berhasil, folder berikut otomatis terbuat di Google Drive Anda dalam folder **ILP 2026**: ``` ILP 2026/ ├── Formulir/ │ ├── Presensi/ │ ├── Pretest/ │ ├── Post Test/ │ ├── Tugas/ │ ├── Laporan Mandiri/ │ └── Feedback/ ├── Materi/ ├── Tugas Peserta/ ├── Data Peserta/ └── Laporan & Analitik/ ``` --- ## Cara Kerja - **Form** dibuat di Google Drive akun Anda (via Forms API) dan diorganisir ke subfolder yang sesuai - **File materi & tugas** diunggah ke Google Drive, tersimpan di subfolder yang tepat - **Respons form** dikirim ke Google Forms dan dicatat di Supabase (data prefill saja) - Semua file di-share "anyone with link" — bisa diakses inline di web via iframe
